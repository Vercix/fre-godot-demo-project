# Godot-Fre Demo Project


## What
<br>

This is a demo project for fre-godot. A fre fork for the Godot engine with javascript bindings.

<br>


## How to use

<br>

Start by installing the packages:

```shell
npm install
``` 

Then build root.
```shell
npm run watch
```


Actual godot project is in project folder.
